가로로 두 칸, 세로로 N칸.
가로로도, 세로로도 붙어있게 배치불가능하다. -> 연속에 대한 문제.

2*N 배열에 사자를 배치하는 경우의 수를 알아보자.

N ( 1 <= N <= 100,000 )
주의)
1) 9901로 나눈 나머지를 출력! 혹시모르니까 long long 으로 하자.

// 맨 밑 행에 둘다 비어있으면 0, 왼쪽이 차있으면 1, 오른쪽이 차있으면 2라 하자.
dp[i][0] = (dp[i-1][0] + dp[i-1][1] + dp[i-1][2]) % MOD;
dp[i][1] = (dp[i-1][0] + dp[i-1][2]) % MOD;
dp[i][2] = (dp[i-1][0] + dp[i-1][1]) % MOD;

결과 : (dp[n][0] + dp[n][1] + dp[n][2]) % MOD;
