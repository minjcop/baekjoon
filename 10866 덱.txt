push_front X : X를 덱의 앞에 넣는다
push_back X : X를 덱의 뒤에 넣는다
pop_front() : 덱의 가장 앞에 있는 수를 빼고, 그 수를 출력한다. 비어있으면 -1
pop_back() : 덱의 가장 뒤에 있는 수를 빼고, 그 수를 출력한다. 비어있으면 -1
size : 덱에 들어있는 정수의 개수
empty : 비어있으면 1, 아니면 0
front : 덱의 가장 앞에 있는 정수를 출력, 비어있으면 -1
back : 덱의 가장 뒤에 있는 정수를 출력, 비어있으면 -1

명령의 수 N(1 <= N <= 10000)
둘째줄부터 명령이 하나씩 주어짐.
정수는 1 <= 정수 <= 100000 

iterator가 각 끝에 2개씩 필요하다.
begin, end가 하나로 부족하고
begin1, end1, begin2, end2가 있어야 한다.(소스 참조)
