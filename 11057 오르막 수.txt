문제)
오름차순을 이루는 수.
인접한 수가 같아도 오름차수. (111222도 오름차수)

수의 길이가 주어졌을 때, 오르막 수의 개수를 구하는 프로그램을 작성하시오.

N ( 1 <= N <= 1000)

주의 ) 
1) 오르막 수의 개수를 10007로 나눈 나머지를 출력한다. 
long long 필요할까 10000 * 10000 = 100000000이니까 int 도 될듯.
2) 수는 0으로 시작할 수 있다.

점화식
dp[n][l] : n은 수의 길이, l은 마지막에 사용된 숫자.
dp[1][0] = dp[1][1] = dp[1][2] = ... = dp[1][9] = 1;


a[n][l] = a[n-1][j] : j : 0 ~ l
